# ONYXNOKAMI

ONYXNOKAMI is a Gen Z anime recommendation chatbot with a dark purple chat interface.

ONYX talks like a chill anime friend and recommends shows based on your mood, vibe, or favorite anime.

## Features

- Anime recommendations by vibe
- Gen Z conversational style
- FastAPI backend
- Groq-powered AI responses
- Simple frontend chat UI
- Secret API key kept in `.env`
